Essentials to Start


Ultimate Disc is a sport that is there for anyone to play. It has many different skill levels and leagues. Ultimate can be picked up at any age and is a good source for a cardio workout. This also helps find new friends and helps with teamwork and collaboration.

  • Cones
  • Disc (must be 175grams for game disc)
  • Light and Dark colored shirts
  • Cleats
  • Optional, Friction Gloves
